## Описание программы
*«Random Quote Generator — это графическое приложение для генерации случайных цитат.  
Позволяет пользователям получать случайные цитаты из предопределенного списка, фильтровать их по автору и теме, а также сохранять историю просмотренных цитат в формате JSON.  
Пользователи могут просматривать историю сгенерированных цитат прямо в приложении.»*

## Требования для запуска
Чтобы запустить приложение, необходимо установить:
- Python 3.6 или выше  
- Стандартная библиотека `tkinter` (обычно входит в поставку Python для Windows и Linux; для macOS — возможно потребуется установка отдельно)  
**Дополнительных зависимостей не требуется.**
